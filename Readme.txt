The Skynet Group

A ML hobby group to have some fun messing around with machine learning and deep neural networks.
 
Using
TensorFlow as the main framework
Python as the programming language
Google Cloud ML as the training and deployment platform

 
1st objective is to be a store of useful information on ML; web links, books, videos and any other form of information.
 
 
2nd objective is to be a collection of ML projects that people are working on by themselves or collaborating  on with others, then share their projects for others in the group to learn from.
 
 
3rd objective is to share events and conferences about ML that members of the group may be interested in attending
